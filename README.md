# Simple AI Chatbot

A minimal chatbot using Hugging Face Transformers.

## Dependencies

- transformers
- torch
- flask
- flask-cors

## Setup

1. Install dependencies:
   pip install -r requirements.txt
   pip install flask flask-cors

2. Run the backend server:
   python src/app.py

3. Open the frontend:
   Open frontend/index.html in your browser
